# Cars Price Prediction

## Overview
This project predicts car prices based on various features such as engine, horsepower, speed, fuel type, and more. The notebook includes data cleaning, exploratory data analysis (EDA), and machine learning modeling using PyCaret.

## Dataset
The dataset used in this project is from Kaggle:  
**Dataset name:** Cars Datasets 2025  
**Source:** [Kaggle](https://www.kaggle.com/abdulmalik1518/cars-datasets-2025)

**Columns include:**
- Company Names
- Cars Names
- Engines
- CC/Battery Capacity
- HorsePower
- Total Speed
- Performance (0 - 100 KM/H)
- Fuel Types
- Seats
- Cars Prices (Target)

## Features
- Data cleaning and handling missing values
- Encoding categorical variables
- Scaling numerical features
- Exploratory Data Analysis (correlations, scatter plots, bar charts)
- Machine learning modeling using PyCaret:
  - Model comparison
  - Model tuning
  - Model evaluation
  - Saving final model
